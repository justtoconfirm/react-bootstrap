# React Boilerplate

[![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Clone Repository

```
$ git clone https://github.com/justtoconfirm/react-bootstrap.git
```

## Install

```
$ cd react-bootstrap
$ npm install
```

## Build

```
# For production
$ npm run build

# For development
$ npm run build:dev
```

## Webpack Bundle Analyzer

```
$ npm run analyzer
```

## Lint

```
$ npm run lint

# Fix issues
$ npm run lint:fix
```
