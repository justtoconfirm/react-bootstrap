## Clone Repository

```
$ git clone https://github.com/justtoconfirm/react-bootstrap.git
```

## Install

```
$ cd react-bootstrap
$ npm install
```

## Build

```
# For production
$ npm run build

# For development
$ npm run build:dev
```

## Storybook

```
$ npm run storybook
```

## Webpack Bundle Analyzer

```
$ npm run analyzer
```

## Lint

```
$ npm run lint

# Fix issues
$ npm run lint:fix
```

## Test

```
$ npm run test
```
